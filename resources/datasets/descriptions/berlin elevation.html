Berlin Elevation

The berlin elevation data set contains a digital elevation model of berlin. The data is in the Tag Image File Format (tiff) and can be used, for example, in cost functions. The original raster graphic was cut to the area of Berlin. The original data can be found under https://www.opendem.info/download_srtm.html.
